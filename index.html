La Comunidad Cloud de Perú

Somos un grupo de apasionados por la nube, liderado por la propia comunidad. Nos reunimos para compartir conocimientos, aprender de las experiencias de otros y construir una red de profesionales sólida. ¡Únete a nuestros meetups y eventos para crecer juntos con AWS!

Nuestra Misión

El AWS User Group Perú es la comunidad oficial para usuarios de Amazon Web Services en el país. Nuestro lema es: "Aprendamos juntos, construyamos juntos, crezcamos juntos". Organizamos meetups, workshops y eventos de networking para todos los niveles.
